```mermaid
graph TD
    %% 스타일 정의
    classDef point fill:#f9f,stroke:#333,stroke-width:2px;

    Start((시작)):::point --> Monitoring["1. 관측성 (Prometheus, Grafana, Micrometer)"]:::monitor

    DistributedTracing --> VirtualThread["3. Java 21+ 고성능 처리 (Virtual Threads)"]:::virtual
    VirtualThread --> Resiliance["4. 장애 전파 방지 (Circuit Breaker - Resilience4j)"]:::reliability
    Resiliance --> CloudNative["5. 클라우드 네이티브 (K8s 연동, Graceful Shutdown)"]:::cloud
    CloudNative --> Performance["6. JVM/DB 튜닝 (GC 최적화, 실행 계획 분석)"]:::reliability

    %% 상세 키워드 연결
    subgraph "5단계 핵심 키워드"
    Monitoring -.-> K1["Metric / Dashboard"]

    Performance -.-> K4["Flame Graph / Heap Dump"]
    end
```
